四个基本概念

### Data 数据

- 存储的基本对象
- 描述事务的符号记录
- 特点：与其**语义**不可区分



- 数据字典
- 数据的解释	

### Database 数据库

- 长期储存在计算机内 有组织 可共享的 大量数据集合
- 存放在管理系统中的内容 数据
- 冗余度较小
- 数据独立性高
- 容易扩展
- 可为各种用户共享

### Database Management System 数据库管理系统

- 数据管理的工具
- 是位于用户与操作系统之间的一层数据管理软件



用途：科学的组织和存储数据 高效地获取和维护数据

GUI or CMD

### Database System 数据库系统

+ = DB + DBMS



使用了数据库的系统

在计算机系统中引入数据库后的系统构成
